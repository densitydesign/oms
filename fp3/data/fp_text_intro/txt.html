Extracting from the Web Pages the Most Frequent and Relevant Textual Entities

We scraped the text of each webpage contained in the web corpus built in the previous phase. The resulting documents kept the categorization of the original web pages. We processed the text corpus using some natural language processing techniques to extract the most frequent and the most relevant text entities these are are assumed as key issues in the discursive space around the socio-economic aspects of FP.

In this way it is possible to identify the most shared as well peculiar issues accordingly to the various categories.
