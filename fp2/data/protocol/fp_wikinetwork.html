harvesting the main article links

To analyse the relationship between the main page and the ones that stemmed from it, Using Wikimedia APIs we wrote a small sketch in processing that, starting from the page “Family Planning” crawls all the “Main article” links.

Kohlschütter, Christian, Peter Fankhauser, and Wolfgang Nejdl. "Boilerplate detection using shallow text features." Proceedings of the third ACM international conference on Web search and data mining. ACM, 2010.

(visual) network analysis

The results are visualized with the software GEPHI, an open-source network analysis and visualization software package, with the use of its force-based algorithm, ForceAtlas2, we produced a networks of the articles connected to “Family planning”.

Sébastien Heymann and Bénédicte Le Grand. “Exploratory Network Analysis: Visualization and Interaction” to appear in Hocine Cherifi (editor), Complex Networks and their Applications, Cambridge University Press. Bastian, Mathieu, Sebastien Heymann, and Mathieu Jacomy. "Gephi: an open source software for exploring and manipulating networks." ICWSM. Jacomy, Mathieu, et al.
