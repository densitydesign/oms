crawling the hyperlinks

We followed the web corpus hyperlinks to discovering other relevant URLs. From this new web corpus have been eliminated all the websites considered too general (i.e facebook, twitter) and the non relevant ones (i.e maps.google, youtube). Furthermore, also the websites with an in-degree index smaller than 3 have been excluded. For this step we used HYPHE a software developed at médialab | SciencesPo. Rogers, Richard. "Mapping public web space with the Issuecrawler." Digital cognitive technologies: Epistemology and the knowledge economy (2010): 89-99."

visual network analysis

We examined through co-link / social networking analysis the network produced with the crawling. Furthermore, we analyzed visually the network using GEPHI, an open-source network analysis and visualization software package. Thanks to its force-based algorithm, ForceAtlas2, we produced the qualitative interpretation of the graph. Sébastien Heymann and Bénédicte Le Grand. “Exploratory Network Analysis: Visualization and Interaction” to appear in Hocine Cherifi (editor), Complex Networks and their Applications, Cambridge University Press. Bastian, Mathieu, Sebastien Heymann, and Mathieu Jacomy. "Gephi: an open source software for exploring and manipulating networks." ICWSM. 2009.
